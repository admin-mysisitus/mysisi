Frontend Developer

Remote / Sidoarjo
Rp 7-10 juta/bulan (tergantung pengalaman)
Full-time 3+ Tahun Pengalaman

Deskripsi Pekerjaan:

  • Mengembangkan interface dan user experience yang menarik menggunakan HTML, CSS, dan JavaScript
  • Optimasi Core Web Vitals dan performance website untuk ranking Google tinggi
  • Berkolaborasi dengan Backend Developer dan UI/UX Designer
  • Melakukan testing dan debugging untuk memastikan kualitas code
  • Code review dan mentoring junior developers

Persyaratan:

  • Minimal 3 tahun pengalaman sebagai Frontend Developer
  • Deep understanding tentang Core Web Vitals & performance optimization
  • Menguasai React.js dan/atau Vue.js dengan advanced concepts
  • Familiar dengan TypeScript dan clean code principles
  • Git version control dan collaborative development
  • Portfolio minimal 5 website commercial projects

Benefits:

  • Flexible working hours & remote work
  • Professional development allowance
  • Health insurance & BPJS coverage
  • Bonus performa bulanan

Remote / Sidoarjo
Rp 4-7 juta/bulan (full-time) | Rp 5-15 juta/project (project-based)
